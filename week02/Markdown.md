# MarkDown 문서 생성

Markdown은 일반 텍스트 기반의 문서 작성 형식으로, 간단한 기호만 사용해 제목, 목록, 링크, 표 등을 표현할 수 있다.  
문법이 단순하여 읽기 쉽고, GitHub와 같은 개발 플랫폼에서 문서화 도구로 널리 사용된다.

## MarkDown 문서 형식의 제안

Markdown 문서는 다음과 같은 순서로 구성하면 이해하기 쉽다.

1. 문서 제목
2. 문서 목적 또는 소개
3. 주요 내용
4. 예제 또는 표
5. 참고 자료

예를 들어 수업 정리 문서는 아래와 같이 작성할 수 있다.

- 문서 제목 작성
- 학습 목표 정리
- 핵심 문법 설명
- 실습 예제 추가
- 마무리 요약

## MarkDown 주요 문법

Markdown은 문서를 쉽게 작성하기 위한 경량 마크업 언어로, #을 이용한 제목, * 또는 -을 이용한 목록, **를 이용한 글씨 강조 등 간단한 기호로 서식을 표현할 수 있다.
또한 링크, 이미지, 코드 블록 등을 간단한 문법으로 작성할 수 있어 GitHub의 README나 기술 문서 작성에 많이 사용된다.
