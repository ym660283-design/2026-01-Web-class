HTML5 기본 구조와 자주 쓰는 태그

이 문서는 HTML5의 기본 구조와 자주 쓰는 태그를 눈으로 확인할 수 있게 작성된 예제입니다.

1) 기본 구조

  • <!DOCTYPE html> : HTML5 문서임을 선언
  • <html> : 문서의 루트 요소
  • <head> : 제목/메타/스타일 정보
  • <body> : 화면에 보이는 내용

2) 자주 쓰는 태그

<h1>~<h6> : 제목 크기 (h1이 제일 큼)

<p> : 문단

<div> : 블록 컨테이너

<span> : 인라인 컨테이너

<a> : 링크

<img> : 이미지

<table> / <tr> / <th> / <td> : 표

3) 예제 태그 배치

제목 태그

h1 제목

h2 제목

h3 제목

h4 제목

h5 제목
h6 제목

문단 태그 span(인라인)을 포함한 예시입니다.

div(블록) 내부에 또 다른 문단과 링크가 있습니다.
