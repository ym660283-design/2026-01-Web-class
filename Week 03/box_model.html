CSS 박스 모델 + 배치 속성 비교

1) 박스 모델: content, padding, border, margin

Box1

content 크기 + padding 8px + border 2px + margin 4px

Box2

content 크기 + padding 16px + border 4px + margin 12px

Box3

content 크기 + padding 24px + border 6px + margin 20px

체크: margin은 밖에 여백, padding은 안쪽 여백, border는 테두리, content는 실제 내용 박스입니다.

2) display: block, inline, inline-block

block은 너비 전체, inline은 콘텐츠만, inline-block은 인라인 흐름이지만 크기 지정 가능.
display: block
display: inline display: inline-block

3) display: flex

flex item 1
flex item 2
flex item 3
flex는 행/열 사이즈 조정, 간격 처리, 정렬을 쉽게 합니다.

4) display: grid

grid item A
grid item B
grid item C

grid는 행/열을 명시적으로 설정하여 2차원 레이아웃을 구성합니다.

5) position: relative + absolute

absolute
relative

position: relative가 있는 부모 박스를 기준으로 absolute 자식이 위치합니다.

절대 위치는 가장 가까운 position 속성이 있는 조상 기준입니다. relative는 원래 위치 기준 이동합니다.
