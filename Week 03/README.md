# Week 03 학습 자료

이 폴더에는 HTML/CSS 실습 예제가 포함되어 있습니다.

## 포함된 파일

- `from_tag.html` - HTML from 태그 예제
- `css_basic.html` - CSS 선택자(태그, 클래스, ID, 어트리뷰트, 하위/자식/형제 선택자)와 색/텍스트 스타일 예제
- `box_model.html` - 박스 모델, display, position 속성 실습
- `flex_display.html` - flex를 이용한 div 레이아웃 vs semantic 태그 레이아웃 비교
- `form_list_table.html` - 리스트, 테이블, 폼 스타일 예제
- `form_list_table.css` - `form_list_table.html`에 연결된 외부 CSS
- `CSS_animation.html` - transition + animation 예제
- `bootstrap_example.html` - Bootstrap CDN 기반 반응형 블로그/대시보드 레이아웃

## 사용 방법

1. 각 HTML 파일을 브라우저에서 열어 결과를 확인합니다.
2. 창 크기를 조절하면서 반응형 레이아웃 동작을 확인합니다.
3. CSS/HTML 코드를 수정하여 스타일 실습을 진행합니다.
