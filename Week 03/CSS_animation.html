CSS Transition & Animation 예제

hover 기반 transition과 @keyframes animation을 동시에 보여줍니다.

Transition 예제

버튼에 transition을 적용하면호버 시 부드럽게 변화합니다.

transition: background 0.25s ease, transform 0.2s ease → 색/크기 변화를 부드럽게.

박스에 배경색과 크기 변화를 적용한 hover transition

Hover
transition: transform 0.3s ease, background-color 0.3s ease → 확대 및 색상 변화.

텍스트에 글자 간격/색을 transition으로 적용

transition text
transition: letter-spacing 0.25s ease, color 0.25s ease → 글자 간격/색상 변화.

Animation 예제

회전 박스에 @keyframes로 무한 회전 효과 적용

Rotate
animation: rotateBox 4s linear infinite → 4초 주기 무한 회전.

페이드 펄스 텍스트에 투명도와 위치 애니메이션

Pulse
animation: fadePulse 2.5s ease-in-out infinite → 페이드 인/아웃 + 위아래 이동.
