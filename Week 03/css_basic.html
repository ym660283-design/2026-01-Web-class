CSS 선택자와 텍스트 스타일 예제

1. 태그 선택자

이 문장은 h1 태그에 적용되는 스타일을 설명하는 기본 예시입니다.

2. 클래스 선택자

이 블록은 .box 클래스에 스타일이 적용됩니다.

ID 선택자 #main-text가 적용된 이 문장은 텍스트 정렬과 라인 높이를 함께 확인할 수 있도록 구성되었습니다.

3. ID 선택자

어트리뷰트 선택자 [data-role="status"]가 이 요소에 적용됩니다.

4. 하위, 자식, 형제 선택자

이 문장은 .module p 하위 선택자에 의해 스타일이 지정되었습니다.

이 문장은 .module > p 자식 선택자에 의해 추가 스타일이 적용됩니다.

이 문장은 h2 + p 바로 다음 형제 선택자에 의해 다르게 표시됩니다.

이 문장은 h2 ~ p 일반 형제 선택자에 의해 스타일이 적용됩니다.

5. 여러 셀렉터로 시각 차이

서로 다른 셀렉터에서 색상과 텍스트 스타일을 분명히 구분했습니다.

같은 HTML 내 여러 셀렉터가 각각 다른 스타일을 적용하는 예를 한눈에 보세요.
