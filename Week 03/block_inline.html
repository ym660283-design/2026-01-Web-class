HTML5 기본 구조 & 자주 쓰는 태그

이 문서는 블록/인라인 태그를 함께 연습하기 위한 학습용 예제입니다.

기본 구조

  • <!DOCTYPE html> : HTML5 선언
  • <html> : 루트 요소
  • <head> : 메타데이터 포함
  • <body> : 콘텐츠 표시

자주 쓰는 태그

<h1>~<h6> : 제목

<p> : 문단

<div> : 블록 컨테이너

<span> : 인라인 컨테이너

<ul> / <ol> / <li> : 목록

<a> : 링크

<img> : 이미지

<table> / <tr> / <th> / <td> : 표

문단과 div/span

이 문장은 <span> 내부의 인라인 텍스트를 포함합니다.

div는 블록 요소로 전체 너비 차지

목록 (ul/ol/li)

  • ul - 순서 없는 목록
  • li - 목록 항목
  1. ol - 순서 있는 목록
  2. li - 목록 항목

링크 + 이미지

이 링크는 Fronty 이미지 태그 변환 설명으로 연결됩니다.
